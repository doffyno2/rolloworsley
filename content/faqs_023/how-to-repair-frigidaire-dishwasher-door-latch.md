<h2>Tools Needed</h2>

<p>You will need a few simple tools to complete this task.  These include:</p>

<ul>
  <li>A Phillips head screwdriver</li>
  <li>A flathead screwdriver</li>
  <li>A pair of needle nose pliers</li>
  <li>A replacement door latch if needed</li>
</ul>

<h2>Step-by-Step Instructions</h2>

<h3>1. Unscrew the Door Latch</h3>

<p>Open the dishwasher door and locate the screws that attach the door latch to the door.  Use a Phillips head screwdriver to remove the screws and then carefully remove the door latch.</p>

<h3>2. Inspect the Latch</h3>

<p>Inspect the door latch for any signs of wear or damage.  If there is no damage, the latch can be tightened using a flathead screwdriver.  If the latch is broken, it will need to be replaced.</p>

<h3>3. Tighten the Latch</h3>

<p>If the door latch is not broken, use a flathead screwdriver to tighten the screws that hold the door latch in place.  Be careful not to over-tighten the screws as this may cause damage to the door.</p>

<h3>4. Replace the Latch</h3>

<p>If the door latch is broken, use a pair of needle nose pliers to remove the broken latch and replace it with a new one.  Be sure to use the same type of screws as the old one and tighten them securely.</p>
